# BPK-016 Review Packet - PR Review Manual Receipts

## Summary

BPK-016 adds a pure manual PR/review receipt normalizer.

- `builder/src/bpkPrReviewManualReceipts.ts` converts operator-supplied PR metadata into
  BPK-012-compatible review records.
- Invalid task IDs, invalid PR URLs, missing commits, invalid review decisions, invalid check
  statuses, and duplicate tasks are quarantined.
- No GitHub API call, PR creation, merge, push, workflow change, package change, runtime change,
  deployment change, or frontend file was added.
