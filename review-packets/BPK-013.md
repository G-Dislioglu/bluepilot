# BPK-013 Review Packet - Runtime Dry-Run Adapter Contract

## Summary

BPK-013 adds a pure runtime dry-run adapter contract.

- `builder/src/runtimeDryRunAdapterContract.ts` converts a BPK-008 runtime integration contract
  into a guarded dry-run invocation plan.
- The plan sets `dryRun:true` and keeps deploy, provider calls, database writes, GitHub writes,
  runtime routes, and runtime dispatch closed.
- Blocked or review-required integration contracts do not become runnable.
- Write-capable integration is blocked from this dry-run-only adapter.
