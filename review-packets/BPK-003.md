# BPK-003 Review Packet - WorkerPacket-to-WLP-Adapter

## Summary

BPK-003 adds a side-effect-free adapter for converting explicit worker packet
data into a WLP-compatible contract draft.

- `builder/src/workerPacketWlpAdapter.ts` exports adapter types and
  `adaptWorkerPacketToWlpContract`.
- The adapter derives `allowed_files` only from explicit worker edit paths plus
  caller-provided governance artifact paths.
- The adapter rejects malformed task metadata, empty edit sets, duplicate edit
  paths, invalid paths, UI tasks without persona, and protected paths.
- No worker dispatch, provider call, GitHub write, runtime route, package file,
  WLP verifier, or orchestrator behavior was changed.
