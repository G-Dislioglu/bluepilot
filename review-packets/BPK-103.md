# BPK-103 Review Packet - Cockpit Patch Authority Review Decision Gate

## Summary

BPK-103 adds a side-effect-free authority-review decision gate for cockpit patch authority-review intake.

- `builder/src/cockpitPatchAuthorityReviewDecisionGate.ts` exports `decideCockpitPatchAuthorityReview`.
- It accepts explicit `approve`, `defer`, or `reject` decisions.
- Only `approve` can become ready, and even then permit issuance, patch application, server mutation, route mutation, and executable actions remain false.
