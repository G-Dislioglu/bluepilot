# BPK-024 Review Packet - Cockpit Live Model Source Decision

## Summary

BPK-024 adds a decision layer for future cockpit live model sourcing.

- `builder/src/cockpitLiveModelSourceDecision.ts` decides whether accepted Live AICOS network data may become a read-only cockpit model source later.
- Live AICOS source readiness requires an accepted network result and operator review reference.
- Sample-only/manual cockpit modes remain distinct from live sourcing.
- The block does not change the cockpit route, renderer, server mount, runtime, or actions.
