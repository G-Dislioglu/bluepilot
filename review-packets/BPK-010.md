# BPK-010 Review Packet - Live AICOS/Card Binding Intake

## Summary

BPK-010 adds a pure AICOS card snapshot intake normalizer.

- `builder/src/aicosCardBindingIntake.ts` validates passed-in card snapshots before they can feed
  card-conditioned dispatch.
- Accepted snapshots become `DispatchConditionCard`-compatible data.
- Invalid, duplicate, unsafe-path, or evidence-less snapshots are quarantined fail-closed with
  deterministic reasons.
- No AICOS call, maya-core call, provider call, runtime route, cache, database write, GitHub write,
  package change, deployment change, or frontend file was added.
