# BPK-005 Review Packet - Pre-Registered Claims

## Summary

BPK-005 adds a pure pre-registered claims gate.

- `builder/src/preRegisteredClaims.ts` checks WLP contract claims against
  explicit registrations.
- Every contract claim must be registered with at least one evidence reference.
- Missing, unexpected, duplicate, malformed, or evidence-less registrations
  block dispatch.
- Review-required or blocked card-conditioned dispatch plans cannot be upgraded
  to allow by claim registration.
- No runtime dispatch, provider call, persistence, GitHub write, server route,
  or frontend was added.
