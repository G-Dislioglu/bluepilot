# BPK-022 Review Packet - Live AICOS Network Connector

## Summary

BPK-022 adds an isolated Live AICOS network connector.

- `builder/src/liveAicosNetworkConnector.ts` exports `fetchLiveAicosCardsThroughIntake`.
- The connector fails closed unless the BPK-015 contract is ready and `liveFetchAllowed`.
- It requires an explicit HTTPS endpoint URL and an auth token provider.
- It fetches JSON through an injected fetcher in tests or default `fetch` at call time.
- It routes fetched payloads through the existing BPK-019 intake adapter.
- It does not mount a route, persist cache, call DB, call GitHub, change cockpit wiring, or change runtime execution.
