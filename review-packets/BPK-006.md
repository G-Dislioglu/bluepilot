# BPK-006 Review Packet - CLI-Deduplizierung / Schema-Generierung

## Summary

BPK-006 adds a deterministic governance-manifest generator for the BPK pre-dispatch line.

- `builder/scripts/generate-bpk-governance-manifest.mjs` reads BPK contracts and emits a stable
  manifest.
- `builder/data/bpk-governance-manifest.json` records deduplicated required commands and schema
  definitions for the new governance artifacts.
- The manifest covers WorkerPacket/WLP draft, card-conditioned dispatch, and pre-registered
  claims.
- The existing builder repo-index generator check is included in the gate so generated artifacts
  stay normalized.
- No package script, dependency, runtime dispatch, server route, provider, GitHub write, or
  frontend was added.
