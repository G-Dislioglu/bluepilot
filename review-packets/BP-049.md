# Review Packet - BP-049

Datum: 2026-05-28
Task: BP-049 - Implement Local Builder Task Contract Candidate Mock

## Ziel

BP-049 baut den ersten lokalen Builder-Task-Contract-Candidate Mock.

Der Mock prueft den BP-048 Handoff deterministisch und erzeugt ein lokales Candidate Output Envelope.

## Was wurde geaendert?

- `tools/builder-task-candidate.cjs` angelegt.
- `tools/test-builder-task-candidate-fixtures.cjs` angelegt.
- `examples/builder-task-candidate/` mit Golden-Fixtures angelegt.
- `contracts/BP-049.json` angelegt.

## Sicherheitsgrenzen

BP-049 macht nicht:

- keinen Live Builder Call,
- kein Builder Task Create,
- kein Execute / Approve / Push / Deploy,
- keine Auth,
- keine Secrets,
- keine DB,
- keine Persistenz,
- keine Ziel-Datei-Writes.

`builder_task_create_allowed` bleibt immer `false`.

`builder_execute_allowed` bleibt immer `false`.

## Erwartetes Verhalten

Der Candidate Mock:

- gibt `candidate_prepared` fuer resolved Scope Resolver Output aus,
- gibt `requires_human_review` fuer Risk Evidence aus,
- gibt `blocked` fuer Scope Resolver Review Status aus,
- gibt `blocked` fuer Task-Create-Intent aus.
