# BPK-031 Review Packet - PR Receipt Artifact Import

## Summary

BPK-031 adds a supplied-data PR receipt artifact importer.

- `builder/src/prReceiptArtifactImport.ts` accepts object or JSON-string artifacts.
- It routes valid artifact data through BPK-027 receipt intake reporting.
