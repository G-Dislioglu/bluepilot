# BPK-020 Review Packet - Runtime Route Mounting Dry Run

## Summary

BPK-020 mounts a default-off runtime dry-run route.

- `builder/src/runtimeDryRunRoute.ts` handles `POST /probe/runtime-dry-run`.
- `builder/src/server.ts` mounts the handler before health fallback.
- The route is closed unless `BLUEPILOT_RUNTIME_DRY_RUN_ROUTE_ENABLED=true`.
- When enabled, it returns a BPK-017 contract response containing a BPK-013 dry-run plan.
- It does not call `orchestrateTask`, providers, DB, GitHub, deploy, live AICOS, or write paths.
