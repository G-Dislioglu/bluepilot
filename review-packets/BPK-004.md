# BPK-004 Review Packet - Card-Conditioned Dispatch

## Summary

BPK-004 adds a pure card-conditioned dispatch planner.

- `builder/src/cardConditionedDispatch.ts` combines a WLP contract draft with
  explicit card snapshots.
- The planner returns `allow`, `review_required`, or `blocked`.
- Missing, invalid, blocked, deprecated, policy-blocked, or path-mismatched
  cards block dispatch.
- Review-only cards downgrade dispatch to review-required.
- No runtime dispatch, provider call, AICOS call, GitHub write, server route, or
  frontend was added.
