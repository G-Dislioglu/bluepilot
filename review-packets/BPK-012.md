# BPK-012 Review Packet - PR/Review Execution Contract

## Summary

BPK-012 adds a pure PR/review execution receipt evaluator.

- `builder/src/bpkPrReviewExecution.ts` evaluates supplied PR metadata against a BPK release
  sequence.
- Missing PRs, invalid PR URLs, commit mismatches, non-green checks, and changes-requested reviews
  block merge readiness.
- Pending reviews keep the receipt in `review_required`.
- No PR is created, no branch is merged, no GitHub API is called, and no workflow/runtime/frontend
  behavior is changed.
